## 详细功能

1. 修改哔哩哔哩视频合集长度
2. 修改视频列表合集标题宽度
3. 解除视频列表标题字符长度限制
4. 修改直播视频界面大小
5. 修改直播分区文字长度限制
6. 修改直播推荐视频距离
7. ~~消除直播痕迹追踪~~
8. 修改~~全部~~评论区间隔
